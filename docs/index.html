        № 01
        
So, just who is this Perris Davis?

        How I got here.
      


      

        

          

            I'm a software engineer. I've worked at Robinhood,
            Hedge, Apple, and lululemon — each
            stop a chance to architect something hard and then ship the code
            that made it real. I'm currently at AWS as a Senior
            Solutions Architect, where I design and implement CI/CD, MDM, and
            AI/ML systems for strategic customers running fleets of Macs on
            Apple Silicon.
          

          

            After hours: a couple of open-source Swift projects
            (napkin, SFSymbolsKit), a color app on the
            App Store (Purizumu), a macOS VM manager
            (Spooktacular), and a writing practice that comes and goes.
            Quietly shipping software since 2015.
          

          

            These days I'm most interested in local-first software, beautiful
            typography on small screens, the unreasonable effectiveness of a well-written
            README, and software that respects the people on the other
            side of the glass.

Selected Work

        Four projects worth showing today.
      


      
    
        
  1. 
          01
          
    
            
    Spooktacular
    
            
    macOS · Open Source · 2026
    
            
    The macOS VM manager I wished existed. Open source for Apple Silicon: ephemeral runner pools, warm-pool scrub validation, Kubernetes orchestration. Two Macs' worth of CI capacity out of one box. 424 tests, MIT-licensed.
    
          
    
          spooktacular.app ↗
        
    2. 

        
  2. 
          02
          
    
            
    Purizumu
    
            
    iOS · Swift · 2024
    
            
    Japanese for "Prism." A color exploration app I built because I got tired of bad color pickers. Tap anywhere to surface a color; assemble palettes grounded in real color theory. Shipping on the App Store.
    
          
    
          App Store ↗
        
    3. 

        
  3. 
          03
          
    
            
    napkin
    
            
    Swift Framework · Open Source · 2026
    
            
    A Swift 6.2 framework for building apps as a tree of isolated, composable units. Uber's RIBs reimagined for Swift Concurrency — the architecture I kept rebuilding in every project until I finally just published it.
    
          
    
          getnapkin.to ↗
        
    4. 

        
  4. 
          04
          
    
            
    SFSymbolsKit
    
            
    Swift Package · Open Source · 2026
    
            
    A tiny Swift package that makes SFSymbols painless. Extensions on String, UIImage, and NSImage; the full symbol catalog generated by a Python script in the repo. Use it; you'll save an afternoon.
    
          
    
          GitHub ↗
